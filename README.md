# WellMate 健康助手应用

## 项目简介
WellMate是一款综合性健康助手应用，集成了身体健康监测和心理健康支持功能，旨在为用户提供全方位的健康管理服务。

## 主要功能

### 1. 身体健康助手（健康助手）
- AI健康对话功能，提供专业健康咨询
- 健康数据展示（步数、睡眠、心率、血压等）

### 2. 心理健康支持（心灵树洞）
- AI心理咨询服务
- 情绪疏导和心理支持
- 消息历史记录和会话管理
- 语音播放功能（可听取AI回复）

### 3. 个人中心（我的）
- 用户信息管理
- 应用设置
- 语言切换（支持英文/中文）
- 会话历史管理

### 4. 总览页面
- 健康数据概览
- 快速访问各项功能

## 技术架构

### 前端技术栈
- **框架**：uni-app x（高性能跨平台应用开发框架）
- **开发语言**：TypeScript (UTS)
- **UI组件**：自定义组件 + uni-app x内置组件
- **国际化**：支持多语言切换（英文/中文）

## 项目结构

```
├── pages/                # 页面组件
│   ├── login/           # 登录页面
│   ├── register/        # 注册页面
│   ├── index/           # 总览页面
│   ├── physical/        # 身体健康助手页面
│   ├── mental/          # 心理健康支持页面
│   ├── mine/            # 个人中心页面
│   └── settings/        # 设置页面
├── utils/               # 工具函数和服务
│   ├── api-service.uts  # API服务封装
│   ├── env-config.uts   # 环境配置
│   └── i18n/            # 国际化相关
├── components/          # 公共组件
├── static/              # 静态资源
├── App.uvue             # 应用入口组件
├── main.uts             # 应用主配置
└── pages.json           # 页面路由配置
```

## 核心功能模块

### 1. AI对话功能
- 支持身体健康和心理健康两种对话类型
- 消息历史记录保存
- 会话管理（创建、查看、删除）
- 语音播放功能（针对AI回复）

### 2. 健康数据管理
- 支持多种健康指标监测
- 第三方健康数据同步
- 数据可视化展示

### 3. 国际化支持
- 自动检测和应用用户语言偏好
- 支持英文和中文切换
- 本地化存储语言设置

## 应用特点

- **跨平台兼容性**：支持iOS、Android、Web等多个平台
- **用户友好界面**：简洁直观的交互设计
- **数据安全**：用户数据保护和隐私安全
- **响应式设计**：适配不同设备屏幕尺寸
